# Sistema de Controle de Documentos

Sistema de gerenciamento de documentos desenvolvido em Python para praticar conceitos de lógica de programação, estruturas de dados e operações CRUD.

## Funcionalidades

- Cadastrar documentos
- Listar documentos
- Buscar documento por ID
- Atualizar documentos
- Remover documentos

## Tecnologias utilizadas

- Python

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Estruturas de dados (listas e dicionários)
- Organização de código em funções
- Tratamento de erros
- Operações CRUD em aplicações de terminal
